📌 Overview

This project is a real-time Sign Language Detection System developed using MediaPipe and OpenCV.
It captures live video from a webcam, detects and tracks hand landmarks, and classifies gestures using a Machine Learning model trained on custom real-time data.

🚀 Features

Real-time hand tracking

Accurate hand landmark detection with MediaPipe

Gesture/sign classification

Training with a custom-built dataset

Live webcam-based prediction

###🧠 Technologies Used

Python

OpenCV

MediaPipe

NumPy

Scikit-learn
